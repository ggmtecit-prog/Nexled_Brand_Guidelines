Color Palette

The NexLed palette should keep every application recognizably technical, modern, and consistent across digital and print use.

How the System Uses Color

  • •

    Green is the core brand color and should lead across identity, emphasis, and primary brand recognition.

  • •

    Blue and red are support colors. Use them intentionally, never as replacements for the green-led system.

  • •

    Neutrals carry typography, surfaces, and layout structure so product and content remain clear.
